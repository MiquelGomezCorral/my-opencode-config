# caveman-help

Quick-reference card. One shot, no mode change.

## What it does

Prints a cheat sheet of Caveman modes, sibling skills, and deactivation triggers. One-shot display; it does not change the active mode or persist anything.

## How to invoke

```
/caveman-help
```

Also triggers on "caveman help", "what caveman commands", "how do I use caveman".

## Example output

```
Modes:
  /caveman              full (default)
  /caveman lite         lighter
  /caveman ultra        extreme
  /caveman wenyan       classical Chinese

Skills:
  /caveman-commit       terse Conventional Commits
  /caveman-review       one-line PR comments
  /caveman-compress     compress a memory file

Deactivate:
  "stop caveman" or "normal mode"
```
